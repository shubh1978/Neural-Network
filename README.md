# Neural Network Experiments – README

## Overview

This project contains basic Neural Network and Python library experiments.
These experiments demonstrate fundamental concepts required for understanding machine learning and neural network models using Python.

---

# List of Experiments

## 1. NumPy Operations

This experiment demonstrates creation of arrays, matrix operations, arithmetic calculations, and transpose functionality using the NumPy library.

**Key Concepts:**

* Array creation
* Mathematical operations
* Matrix handling
* Transpose operation

---

## 2. Pandas Data Handling

This experiment demonstrates creating a DataFrame from structured data and performing statistical operations like mean calculation.

**Key Concepts:**

* DataFrame creation
* Data manipulation
* Statistical analysis

---

## 3. SciPy Integration

This experiment demonstrates numerical integration of mathematical functions using SciPy scientific computing tools.

**Key Concepts:**

* Function definition
* Numerical integration
* Scientific computation

---

## 4. Matplotlib Visualization

This experiment demonstrates plotting graphs to visualize relationships between variables using Matplotlib.

**Key Concepts:**

* Graph plotting
* Data visualization
* Axis labeling

---

## 5. Perceptron Model

This experiment demonstrates implementation of a Perceptron to solve logical AND gate classification problem.

**Key Concepts:**

* Single-layer neural network
* Binary classification
* Supervised learning

---

## 6. XOR Problem using MLP

This experiment demonstrates solving the XOR logical problem using Multi-Layer Perceptron since single-layer models cannot solve it.

**Key Concepts:**

* Nonlinear classification
* Hidden layers
* Multi-layer learning

---

## 7. Activation Functions Visualization

This experiment demonstrates plotting Sigmoid, Tanh, and ReLU activation functions to understand neural activation behavior.

**Key Concepts:**

* Sigmoid function
* Tanh function
* ReLU function

---

## 8. Multi-Layer Perceptron (MLP)

This experiment demonstrates building an MLP model to classify nonlinear input patterns.

**Key Concepts:**

* Hidden layers
* Activation functions
* Neural network training

---

## 9. Optimizers (Adam & Gradient Descent)

This experiment demonstrates training neural networks using optimization algorithms like Adam to minimize loss.

**Key Concepts:**

* Loss minimization
* Gradient descent
* Adam optimizer

---

## 10. Convolutional Neural Network (CNN)

This experiment demonstrates building CNN architecture used in image processing tasks.

**Key Concepts:**

* Convolution layers
* Pooling layers
* Feature extraction

---

## 11. Recurrent Neural Network (RNN)

This experiment demonstrates sequence processing using recurrent neural networks.

**Key Concepts:**

* Sequential learning
* Memory retention
* Time-series processing

---

## 12. Hopfield Network

This experiment demonstrates associative memory storage and retrieval using Hopfield networks.

**Key Concepts:**

* Pattern storage
* Pattern recall
* Associative memory

---

# Requirements

Install required Python libraries before running the experiments:

```bash
pip install numpy pandas scipy matplotlib scikit-learn tensorflow
```

# How to Run

1. Install dependencies.
2. Open Jupyter Notebook or Google Colab.
3. Copy each experiment code into separate cells.
4. Execute the cells sequentially.
5. Observe outputs and graphs.
